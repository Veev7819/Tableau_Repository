## Tableau Dashboard Portfolio
### Overview

This repository contains a collection of **Tableau dashboards** built to analyze real-world business and workforce problems using data visualization and analytical storytelling.

Each project focuses on transforming raw data into **actionable insights** that support decision-making across HR, freelance marketplaces, and organizational performance.

### Projects
### 1. Human Resources Analytics

**Focus:** Workforce size, demographics, attrition trends, location concentration, and salary insights.

**Key Highlights:**
- Hiring vs termination trends over time
- Department-level workforce distribution
- Demographic insights by age, education, and gender
- Location concentration and salary analysis
